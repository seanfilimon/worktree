The **W0rkTree SDK** (`worktree-sdk`) provides a high-level Rust API for embedding W0rkTree capabilities in your own applications. It powers both the CLI and the Admin Panel, ensuring feature parity across all interfaces.

<Callout type="info">
The SDK API is not yet stable during the `0.x` series. Pin your version and review changelogs before upgrading.
</Callout>

## Overview

Use the SDK to:

- **Manage repositories** programmatically — create, open, and configure
- **Perform version control operations** — commit, branch, diff, merge
- **Build editor integrations** — VS Code, IntelliJ, Neovim plugins
- **Automate CI/CD workflows** — custom pipeline steps with full W0rkTree access
- **Create monitoring tools** — real-time dashboards and health checks
- **Manage multi-tenancy** — create, configure, and isolate tenants
- **Enforce licensing** — assign SPDX licenses and issue grants
- **View staged visibility** — track in-flight changes and real-time edits
- **Administer access** — configure IAM roles and declarative access
