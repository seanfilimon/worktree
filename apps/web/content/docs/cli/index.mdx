The `wt` command-line interface is the primary way to interact with W0rkTree. This section documents every available command, grouped by category.

<Callout type="info">
All commands support the global flags listed below. Command-specific flags are documented under each command.
</Callout>

## Global Options

| Flag | Short | Description |
|---|---|---|
| `--verbose` | `-v` | Enable verbose output with debug information |
| `--quiet` | `-q` | Suppress non-essential output |
| `--config <path>` | `-c` | Path to a custom config file |
| `--no-color` | | Disable colored output |
| `--json` | | Output results as JSON (useful for scripting) |
| `--help` | `-h` | Show help for any command |
| `--version` | `-V` | Print the current W0rkTree version |
